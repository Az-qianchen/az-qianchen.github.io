# 🛠️工具软件
## 📼 前言

- 行走互联网不免会需要用到一些称手的兵器，也正是这些工具赋予了冰冷硬件灵魂，才使其发挥出强大的功能
- 推荐的大部分软件都会以开源为主，故此也做到了真正的绿色、免费、无广，不过相应功能也很难有商业软件那般全面，但已能满足大部分基础需求
- 将代码仓库放在 GitHub 上的软件，我通过徽章显示了其 Stars 数量，对于不明白 Stars 数量含义的小伙伴，可以理解为收藏数，越多也意味着其越受欢迎
